# flask-rest-companies-ner
Flask REST API micro-service with a single resource ` ScanText ` for extracting company names and confidence scores from a given piece of text.

### Requirements:
- torch (>=1.9.0)
- transformers (>=4.8.2)
- flask (>=1.1.2)
- flask-restful (>=0.3.9)


### Run API Server: 

```
python app.py
```

or, 

### Run client Tests:

```
pytest
```
